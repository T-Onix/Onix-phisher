```

``` 
- Clone this repository 
```
Git clone https://github.com/T-Onix/Onix-phisher.git
```
- Then go to cloned directory
```
cd Onix_Phisher
```
- install requirements
```
pip install -r requirements.txt
```
- Finally run the program
```
python Onix.py
```
- For linux
```
sudo python Onix.py
```
